# Aplikasi Penerjemah Indonesia - Inggris - Arab

Aplikasi web sederhana untuk menerjemahkan bahasa Indonesia ke Inggris dan Arab secara online.

## Fitur

- 🌐 Terjemahan Indonesia ↔ Inggris
- 🌍 Terjemahan Indonesia ↔ Arab
- 🔊 Text-to-speech untuk ketiga bahasa
- 📋 Copy teks hasil terjemahan
- ⌨️ Shortcut keyboard (Ctrl+Enter untuk terjemah, Escape untuk hapus)
- 📱 Responsive design untuk mobile

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (Vanilla)
- [MyMemory Translation API](https://mymemory.translated.net/) (gratis)

## Cara Menggunakan

1. Buka halaman aplikasi
2. Masukkan teks bahasa Indonesia di kolom input
3. Klik tombol "Terjemahkan" atau tekan Ctrl+Enter
4. Hasil terjemahan akan muncul dalam bahasa Inggris dan Arab
5. Gunakan tombol speaker untuk mendengarkan pengucapan
6. Gunakan tombol copy untuk menyalin teks
